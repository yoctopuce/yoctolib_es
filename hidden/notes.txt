Nouvelle lib EcmaScript6+:
---------------------------

Babel: support EcmaScript 6 (+ async ES7)

Browserify: interprète syntaxe require() comme en Node.JS => combinée en un seul fichier
----> https://www.npmjs.com/package/babelify = Browserify avec Babel intégré

Uglify - minification (eventuellement intégré à Browserify: minifyify).
  Attention, sourcemaps fonctionnent avec minifyify, mais step-by-step pas forcément !

EsLint - Verifications sémantiques

Gulp: chaînage des tâches (aka Make), build facilité

websocket/ws - support WebSocket dans Node.JS

ExpressJS - support server-side serveur HTTP dans Node.JS

(npm: package manager)


------------------------------------
Environment setup:

npm install -g gulp             - Installation globale sur la machine

Puis pour pouvoir générer le projet:
npm install

Pour info: les commandes à faire manuellement pour ajouter
les dépendances au projet et qu'il s'installe tout seul avec
npm install:
    npm install --save-dev gulp
    npm install --save-dev gulp-babel
    npm install --save-dev gulp-sourcemaps
    npm install --save-dev babel-preset-es2015
    npm install --save-dev babel-preset-stage-3
    npm install --save-dev eslint@latest
    npm install --save-dev babel-eslint@latest
